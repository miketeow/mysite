# Architecture

This project is designed as a modular monolith that can be split into microservices.

## Key Features
* **Auth:** JWT-based authentication with Refresh Tokens.
* **Caching:** Redis is used to cache product details, reducing DB load by 80%.
* **Rate Limiting:** Custom middleware to prevent abuse.

# Deployment

The entire stack is containerized using Docker Compose for easy local development.

```yaml
services:
  api:
    build: .
    ports:
      - "3000:3000"
  redis:
    image: redis:alpine
```
