# Real-Time Sync

The core feature of this app is real-time collaboration. I used **Supabase Realtime** (Postgres Changes) to listen for database updates.

When User A moves a card:
1.  Update is sent to Supabase.
2.  Supabase broadcasts the change.
3.  User B's UI updates instantly without refreshing.

# Database Schema

I used Drizzle ORM for type-safe database queries.

```typescript
export const tasks = pgTable('tasks', {
  id: serial('id').primaryKey(),
  title: text('title').notNull(),
  status: text('status').default('todo'),
});
```
