### JavaScript

A simple JavaScript snippet demonstrating DOM manipulation and an event listener.

```javascript
// Select an element
const button = document.querySelector('#my-button');

/**
 * Handles the click event.
 * @param {Event} event - The click event.
 */
function handleClick(event) {
  event.preventDefault();
  console.log('Button clicked!', event.target);

  const message = "Hello, world!";
  alert(message);
}

// Add the listener
if (button) {
  button.addEventListener('click', handleClick);
}
```

### TypeScript
Here we test TypeScript-specific syntax like interfaces, type annotations, and generics.


```ts
interface User {
  id: number;
  username: string;
  isAdmin?: boolean; // Optional property
}

function getUser<T extends User>(users: T[], id: number): T | undefined {
  return users.find(user => user.id === id);
}

const users: User[] = [
  { id: 1, username: 'alice', isAdmin: true },
  { id: 2, username: 'bob' },
];

const user = getUser(users, 1);

if (user) {
  console.log(`Found user: ${user.username.toUpperCase()}`);
}
```

### Rust

Rust has a unique syntax with macros, lifetimes, and specific keywords.

```rust
#[derive(Debug)]
struct Point {
    x: i32,
    y: i32,
}

impl Point {
    // A static method (associated function)
    fn new(x: i32, y: i32) -> Self {
        Self { x, y }
    }

    // A method that borrows self
    fn distance_from_origin(&self) -> f64 {
        let sum_squares = (self.x.pow(2) + self.y.pow(2)) as f64;
        sum_squares.sqrt()
    }
}

fn main() {
    let p = Point::new(3, 4);
    println!("Created point: {:?}", p);
    println!("Distance from origin: {:.2}", p.distance_from_origin());

    let message = String::from("Hello from Rust!");
    // Test a closure
    let print_message = || println!("{}", message);
    print_message();
}
```

### C

Finally, a classic C example to test low-level language highlighting, including preprocessor directives, pointers, and standard library functions.

```c
#include <stdio.h>
#include <stdlib.h>

// A simple struct
typedef struct {
    int x;
    int y;
} Point;

// Function that takes a pointer to a struct
void print_point(Point *p) {
    if (p == NULL) {
        return;
    }
    printf("Point at (%d, %d)\n", p->x, p->y);
}

int main(int argc, char *argv[]) {
    printf("Hello, World! Program name: %s\n", argv[0]);

    // Allocate memory for a point
    Point *p1 = (Point *)malloc(sizeof(Point));
    if (p1 == NULL) {
        fprintf(stderr, "Memory allocation failed\n");
        return 1;
    }

    p1->x = 10;
    p1->y = 20;

    print_point(p1);

    // Free the allocated memory
    free(p1);
    p1 = NULL;

    return 0;
}
```
